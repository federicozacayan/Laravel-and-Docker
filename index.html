Laravel and Docker

Follow the oficcial documentation: https://laradock.io/getting-started/

git clone https://github.com/laradock/laradock.git

Go to the laradock folder

cd laradock/

Copy the environment variables.

cp env-example .env

Modify the project folder into .env file.

APP_CODE_PATH_HOST=../project-z/

Run the container.

sudo docker-compose up -d nginx mysql

Go into the container.

sudo docker-compose exec workspace bash

Install Laravel with composer.

composer create-project --prefer-dist laravel/laravel . "5.7.*"

Grant permission to the folder project.

sudo chmod 777 -R upload-file/

Test the application in the browser.

http://localhost/
