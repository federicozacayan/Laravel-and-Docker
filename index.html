Laravel and Docker

Follow the oficcial documentation: https://laradock.io/getting-started/

git clone https://github.com/laradock/laradock.git

Go to the laradock folder

cd laradock/

Copy the environment variables.

cp env-example .env

Modify the project folder into .env file.

APP_CODE_PATH_HOST=../project-z/

Run the container.

sudo docker-compose up -d nginx mysql

Go into the container.

sudo docker-compose exec workspace bash

Install Laravel with composer.

composer create-project --prefer-dist laravel/laravel . "5.7.*"

Grant permission to the project folder.

sudo chmod -R 777  .

Test the application in the browser.

http://localhost/

Set database

Copy the values in /project-z/.env (or in /var/www/.env if you are into the container) from laradock/.env.

/* laradock/.env */
...
MYSQL_DATABASE=default
MYSQL_USER=default
MYSQL_PASSWORD=secret
...
/* project-z/.env */
...
DB_HOST=mysql
...
DB_DATABASE=default
DB_USER=default
DB_PASSWORD=secret
...

Check the mysql container name (probably 'laradock_mysql_1'), and go into it.

sudo docker exec -it laradock_mysql_1 bash

Once inside, connect the database using mysql client.

root@randon123:/# mysql -u root -p

Use the password in located in php-project/laradock/.env, which is root.

MYSQL_ROOT_PASSWORD=root

Then, you must modify the form of identification that the user uses.

mysql> ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'root';
Query OK, 0 rows affected (0.08 sec)

mysql> ALTER USER 'root'@'%' IDENTIFIED WITH mysql_native_password BY 'root';
Query OK, 0 rows affected (0.11 sec)

mysql> ALTER USER 'default'@'%' IDENTIFIED WITH mysql_native_password BY 'secret';
Query OK, 0 rows affected (0.14 sec)
